# 使用说明

除了前端的H5C3的笔记用的是excl表格整理的（那时并没学MD语法），其他的笔记都使用了MD语法来撰写，如果想了解MD语法如何使用，请查看工具文件夹里的markdown文件夹。

其中有些笔记有些简陋或者格式混乱，是因为我对那项技术并没有全面学习或只是学习了校内课程，请原谅。
